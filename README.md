# Financial_Fraud_Detection_PowerBI
A Credit Card Analysis dashboard developed using Power BI to explore transaction behavior, spending patterns, and potential risk indicators. The project focuses on transforming raw financial data into meaningful insights through interactive visualizations, KPI tracking, and trend analysis.


## Project Overview
This project presents a Financial Fraud Detection dashboard developed using Power BI. The dashboard analyzes credit card transaction data to uncover spending patterns, customer behavior, trends, and potential fraud indicators through interactive visualizations.

## Dashboard Features
- Transaction behavior analysis  
- Spending pattern visualization  
- Fraud risk indicators  
- KPI tracking & trend analysis  
- Interactive filters & slicers  

## Tools & Technologies Used
- **Power BI** – Data visualization & dashboard creation  
- **Dataset** – Credit Card Transaction Data  
- **Data Processing** – Power Query (Data Cleaning & Transformation)
  
## Dataset Description
The dataset contains credit card transaction details used to analyze:

- Transaction amounts  
- Customer activity  
- Spending trends  
- Anomaly / fraud patterns  

## Key Insights
The dashboard helps identify:

✔ Unusual transaction behavior  
✔ High-risk spending patterns  
✔ Transaction trends  
✔ Potential fraud indicators  

## Dashboard Preview
The interactive Power BI dashboard can be accessed by downloading and opening the `.pbix` file included in this repository.

Open the file using **Power BI Desktop** to explore the full dashboard experience.

## How to Use
1. Download the `.pbix` file  
2. Open using **Power BI Desktop**  
3. Explore dashboard using filters & visuals
